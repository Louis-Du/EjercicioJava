# Ejercicios de Consola

## Descripción

Esta carpeta contiene ejercicios de programación en Java ejecutados en línea de comandos. Los proyectos se enfocan en practicar conceptos fundamentales del lenguaje mediante aplicaciones interactivas que utilizan entrada/salida estándar.

## Conceptos Practicados

Los ejercicios en esta sección cubren:

- Entrada y salida de datos mediante `Scanner`
- Estructuras de control (condicionales, bucles)
- Validación de entrada de usuario
- Manejo de arreglos y colecciones (`ArrayList`, `HashMap`)
- Algoritmos básicos (búsqueda, filtrado, cálculos)
- Manipulación de cadenas de texto
- Generación de números aleatorios
- Organización de código en métodos y funciones

## Proyectos

1. **01 - NumeroPrimos**: Generación y verificación de números primos
2. **02 - AdivinarNumero**: Juego de adivinanza con validación de entrada
3. **03 - AnalisisTexto**: Análisis estadístico de cadenas de texto
4. **04 - ListadorNumeros**: Sistema de filtrado y almacenamiento de números
5. **05 - informeVenta**: Generación de informes de ventas con matrices
6. **06 - manejoProductos**: Sistema básico de gestión de productos

## Ejecución

Todos los programas se ejecutan desde la línea de comandos utilizando:

```bash
javac NombreDelArchivo.java
java NombreDelArchivo
```

Cada proyecto incluye instrucciones interactivas en la consola para guiar al usuario.
