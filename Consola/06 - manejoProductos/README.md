# Sistema de Manejo de Productos

## Tipo
Consola

## Descripción

Sistema básico de gestión de productos que implementa operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre un catálogo de productos. Permite al usuario administrar un inventario mediante un menú interactivo de opciones.

## Conceptos Practicados

- Programación orientada a objetos (clase `Producto`)
- Colecciones de objetos (`ArrayList<Producto>`)
- Operaciones CRUD en colecciones
- Menú interactivo con bucle principal
- Búsqueda y modificación de elementos en listas
- Métodos de instancia y encapsulación
- Entrada y validación de datos estructurados
- Manejo de índices y posiciones en listas

## Funcionamiento

El programa presenta un menú con las siguientes opciones:

1. **Agregar Producto**: Permite crear un nuevo producto con sus atributos
2. **Listar Productos**: Muestra todos los productos registrados
3. **Actualizar Producto**: Modifica los datos de un producto existente
4. **Eliminar Producto**: Remueve un producto del catálogo
5. **Salir**: Finaliza la ejecución del programa

Cada producto contiene información como nombre, precio, cantidad y otros atributos relevantes según la implementación.

## Notas

Este ejercicio combina conceptos de programación orientada a objetos con manipulación de colecciones, demostrando un patrón común en sistemas de gestión de datos. La estructura del código separa la lógica de la clase `Producto` de la lógica de manejo del catálogo.
