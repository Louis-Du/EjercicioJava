# Números Primos

## Tipo
Consola

## Descripción

Programa que genera y verifica números primos. Solicita al usuario la cantidad de números primos que desea generar y muestra una lista con los primeros N números primos encontrados.

## Conceptos Practicados

- Entrada y salida de datos con `Scanner`
- Implementación de funciones (métodos estáticos)
- Algoritmos de verificación de números primos
- Uso de arreglos para almacenamiento
- Estructuras de control (bucles `while` y `for`)
- Operadores aritméticos y de módulo
- Formateo de salida

## Funcionamiento

El programa implementa una función `esPrimo()` que verifica si un número es primo dividiendo entre todos los números menores que él. Una vez identificado un número primo, se almacena en un arreglo hasta completar la cantidad solicitada por el usuario.
