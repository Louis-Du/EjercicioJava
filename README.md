# EjercicioJava

## Descripción

Este repositorio contiene una colección de ejercicios prácticos desarrollados en Java con el propósito de documentar y consolidar el aprendizaje del lenguaje. Los proyectos abarcan desde aplicaciones de consola hasta interfaces gráficas utilizando Swing y AWT.

## Propósito Educativo

Los ejercicios incluidos provienen de diversas fuentes de aprendizaje:

- Cursos formales de programación
- Programas de formación tecnológica
- Estudio autodidacta y práctica personal

El objetivo principal es mantener un registro organizado del progreso en el desarrollo de habilidades de programación en Java.

## Estructura del Repositorio

El repositorio está organizado en tres carpetas principales:

### Consola

Contiene ejercicios ejecutados en línea de comandos que practican conceptos fundamentales como entrada/salida, estructuras de control, manejo de colecciones, validación de datos y algoritmos básicos.

**Proyectos:**
- 01 - NumeroPrimos
- 02 - AdivinarNumero
- 03 - AnalisisTexto
- 04 - ListadorNumeros
- 05 - informeVenta
- 06 - manejoProductos

### Applet

Aplicaciones con interfaz gráfica desarrolladas inicialmente como Java Applets y posteriormente adaptadas usando Swing. Incluyen proyectos interactivos con entrada de datos y cálculos dinámicos.

**Proyectos:**
- AnalizadorDeTriangulos
- ConvertidorDias
- GeneradorDeFacturas

### Frame

Ejercicios prácticos enfocados en el desarrollo de interfaces gráficas utilizando componentes de Swing y AWT. Cada proyecto explora diferentes elementos de interfaz, layouts, eventos y modelos de datos.

**Proyectos:**
- Componentes básicos (botones, menús, barras de herramientas)
- Controles de entrada (cuadros de texto, verificación, listas, combos)
- Layouts y diseños personalizados
- Eventos y manejo de interacciones
- Diálogos y componentes avanzados

## Tecnologías Utilizadas

- **Java SE**: Lenguaje principal de desarrollo
- **Swing**: Framework para interfaces gráficas
- **AWT**: Abstract Window Toolkit para componentes UI

## Notas

Este repositorio tiene una naturaleza educativa y se encuentra en continuo crecimiento conforme se desarrollan nuevos ejercicios y proyectos de aprendizaje. Los ejercicios están documentados con comentarios explicativos en el código fuente.
