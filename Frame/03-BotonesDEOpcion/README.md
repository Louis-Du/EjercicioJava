# Botones de Opción

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que explora el uso de botones de opción (JRadioButton) en Java Swing. Los radio buttons permiten al usuario seleccionar una única opción de un conjunto mutuamente exclusivo.

## Conceptos Practicados

- Componente JRadioButton
- Uso de ButtonGroup para agrupación
- Selección mutuamente exclusiva
- Manejo de eventos de selección
- Verificación del estado de botones (isSelected)
- Diseño de formularios con opciones únicas

## Funcionamiento

La aplicación presenta varios botones de opción agrupados. El usuario puede seleccionar solo un botón a la vez dentro de cada grupo. Al seleccionar un nuevo botón, el anterior se deselecciona automáticamente.

## Notas

JRadioButton es ideal para situaciones donde el usuario debe elegir exactamente una opción de un conjunto limitado de alternativas. ButtonGroup asegura que solo un botón esté seleccionado a la vez.
