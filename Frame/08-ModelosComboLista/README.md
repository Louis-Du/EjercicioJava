# Modelos de ComboBox y Lista

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio avanzado que explora el uso de modelos de datos personalizados para JComboBox. Demuestra cómo implementar ComboBoxModel para controlar el contenido y comportamiento de listas desplegables.

## Conceptos Practicados

- Interfaz ComboBoxModel
- Implementación de DefaultComboBoxModel
- Separación de datos y presentación (patrón MVC)
- Actualización dinámica de datos
- Notificación de cambios a listeners
- Manipulación programática de elementos

## Funcionamiento

La aplicación utiliza modelos personalizados para gestionar los datos mostrados en combo boxes. Los modelos permiten agregar, eliminar y modificar elementos dinámicamente, con actualización automática de la interfaz.

## Notas

El uso de modelos proporciona mayor flexibilidad y permite implementar lógica compleja de gestión de datos separada de la presentación visual.
