# Vectores y Eventos

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Proyecto integrador que combina el manejo de vectores (colecciones dinámicas) con el sistema de eventos de Swing. Demuestra cómo crear aplicaciones interactivas que responden a eventos del usuario mientras gestionan datos en estructuras dinámicas.

## Conceptos Practicados

- Integración de vectores con eventos
- Manejo de colecciones dinámicas
- Respuesta a eventos de usuario
- Actualización sincronizada de datos y UI
- Componente JTable con eventos
- Manejo de eventos de selección
- Validación y procesamiento de datos
- Patrón Observer en interfaces gráficas

## Funcionamiento

La aplicación gestiona datos en vectores y responde a eventos de usuario (clics de botón, selecciones, cambios de texto) para modificar los datos. La interfaz se actualiza automáticamente para reflejar los cambios en la estructura de datos subyacente.

## Notas

Este proyecto demuestra un patrón fundamental en aplicaciones interactivas: la coordinación entre el modelo de datos (vectores) y la vista (componentes Swing) mediante el sistema de eventos. Es esencial para aplicaciones que manejan datos dinámicos con interacción del usuario.
