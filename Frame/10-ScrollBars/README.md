# Barras de Desplazamiento

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que demuestra el uso de barras de desplazamiento (JScrollBar) en Java Swing. Los scrollbars permiten ajustar valores dentro de un rango definido mediante desplazamiento.

## Conceptos Practicados

- Componente JScrollBar
- Orientación horizontal y vertical
- Manejo de eventos de ajuste (AdjustmentListener)
- Configuración de rangos (mínimo, máximo)
- Configuración de incrementos
- Obtención del valor actual (getValue)
- Sincronización con otros componentes

## Funcionamiento

La aplicación presenta barras de desplazamiento que el usuario puede manipular arrastrando el deslizador o haciendo clic en las flechas. Los cambios en la posición del scrollbar generan eventos que pueden utilizarse para actualizar otros componentes o realizar acciones.

## Notas

JScrollBar puede usarse independientemente para control de valores o como parte de JScrollPane para proporcionar navegación en contenido que excede el área visible.
