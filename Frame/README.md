# Ejercicios de Interfaces Gráficas (Frame)

## Descripción

Esta carpeta contiene ejercicios prácticos enfocados en el desarrollo de interfaces gráficas utilizando componentes de Swing y AWT. Cada proyecto explora diferentes elementos de interfaz de usuario, permitiendo aprender de forma incremental el uso de componentes, layouts, eventos y modelos de datos en Java.

## Conceptos Practicados

Los ejercicios en esta sección cubren:

- Componentes básicos de Swing (JButton, JLabel, JTextField, JTextArea)
- Controles de selección (JCheckBox, JRadioButton, JComboBox, JList)
- Componentes de navegación (JMenu, JMenuBar, JToolBar)
- Diálogos y ventanas emergentes (JDialog, JOptionPane, JFileChooser)
- Layouts (BorderLayout, FlowLayout, GridLayout)
- Componentes avanzados (JSlider, JSpinner, JScrollBar, JScrollPane)
- Manejo de eventos (ActionListener, MouseListener)
- Modelos de datos (ListModel, ComboBoxModel)
- Vectores y tablas dinámicas

## Ejecución

La mayoría de los proyectos están estructurados como proyectos de NetBeans y pueden ejecutarse:

1. Abriendo el proyecto en NetBeans u otro IDE compatible
2. Ejecutando la clase principal que contiene el método `main`
3. Compilando y ejecutando desde línea de comandos

Cada proyecto presenta una ventana gráfica (JFrame) que demuestra el uso del componente específico.

## Notas

Los proyectos siguen una progresión didáctica, comenzando con componentes simples y avanzando hacia implementaciones más complejas. Es recomendable revisar los ejercicios en orden numérico para comprender mejor la evolución de los conceptos.
