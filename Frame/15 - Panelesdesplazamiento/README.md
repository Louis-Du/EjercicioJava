# Paneles de Desplazamiento

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que explora el uso de paneles con desplazamiento (JScrollPane) en Java Swing. Los scroll panes proporcionan barras de desplazamiento automáticas para contenido que excede el área visible.

## Conceptos Practicados

- Componente JScrollPane
- Envoltura de componentes con scroll
- Configuración de políticas de scroll (horizontal/vertical)
- Barras de desplazamiento automáticas
- Viewport y área visible
- Integración con JTextArea, JList, JTable
- Manejo de contenido de gran tamaño

## Funcionamiento

La aplicación presenta componentes envueltos en JScrollPane que muestran barras de desplazamiento cuando el contenido excede el espacio disponible. El usuario puede usar las barras de desplazamiento, rueda del ratón o arrastrar para navegar por el contenido.

## Notas

JScrollPane es fundamental para hacer que componentes de tamaño variable o grande sean navegables. Se usa comúnmente con áreas de texto, listas y tablas.
