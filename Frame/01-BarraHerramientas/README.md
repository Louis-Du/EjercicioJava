# Barra de Herramientas

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio práctico que demuestra el uso del componente JToolBar en Java Swing. Implementa una barra de herramientas con botones que pueden contener iconos y texto, permitiendo acceso rápido a funcionalidades comunes de la aplicación.

## Conceptos Practicados

- Componente JToolBar
- Uso de JButton dentro de barras de herramientas
- Carga y visualización de iconos (ImageIcon)
- Manejo de eventos con ActionListener
- Configuración de propiedades de botones (focusable, posición de texto)
- Diseño de interfaz con barra de menú y barra de herramientas

## Funcionamiento

La aplicación presenta una ventana con una barra de herramientas superior que contiene varios botones. Los botones pueden mostrar iconos, texto, o ambos, y responden a eventos de clic del usuario.

## Notas

JToolBar es un componente esencial en aplicaciones de escritorio profesionales, proporcionando acceso rápido a las funcionalidades más utilizadas.
