# Menús Emergentes

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que demuestra el uso de menús contextuales (JPopupMenu) en Java Swing. Los menús emergentes aparecen cuando el usuario hace clic derecho sobre un componente, proporcionando opciones contextuales.

## Conceptos Practicados

- Componente JPopupMenu
- Manejo de eventos del ratón (MouseListener)
- Detección de clic derecho (isPopupTrigger)
- Posicionamiento dinámico de menús
- Creación de acciones contextuales
- Integración con otros componentes

## Funcionamiento

La aplicación responde a eventos de clic derecho del ratón mostrando un menú emergente en la posición del cursor. El menú contiene opciones específicas al contexto del componente sobre el que se hizo clic.

## Notas

JPopupMenu es fundamental para proporcionar acceso rápido a funcionalidades relacionadas con el contexto actual. Es un patrón común en aplicaciones de escritorio modernas.
