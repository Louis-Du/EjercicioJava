# Control Giratorio (Spinner)

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que demuestra el uso de controles giratorios (JSpinner) en Java Swing. Los spinners permiten al usuario incrementar o decrementar valores mediante botones de flecha o entrada directa.

## Conceptos Practicados

- Componente JSpinner
- Modelos de spinner (SpinnerModel)
- SpinnerNumberModel para valores numéricos
- SpinnerDateModel para fechas
- SpinnerListModel para listas de valores
- Manejo de eventos de cambio (ChangeListener)
- Configuración de rangos y pasos de incremento

## Funcionamiento

La aplicación presenta uno o más spinners que permiten al usuario ajustar valores usando botones de flecha arriba/abajo o escribiendo directamente. Los spinners pueden configurarse para trabajar con números, fechas o listas personalizadas de valores.

## Notas

JSpinner es ideal para entrada de valores numéricos con validación automática. Combina la precisión de entrada directa con la facilidad de uso de controles de incremento/decremento.
