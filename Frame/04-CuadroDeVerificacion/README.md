# Cuadro de Verificación

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que demuestra el uso de casillas de verificación (JCheckBox) en Java Swing. Los checkboxes permiten al usuario seleccionar múltiples opciones de forma independiente.

## Conceptos Practicados

- Componente JCheckBox
- Selección múltiple independiente
- Manejo de eventos (ItemListener, ActionListener)
- Verificación del estado (isSelected)
- Diseño de formularios con opciones múltiples
- Procesamiento de selecciones múltiples

## Funcionamiento

La aplicación presenta varias casillas de verificación que el usuario puede marcar o desmarcar de forma independiente. Cada checkbox mantiene su propio estado sin afectar a los demás.

## Notas

JCheckBox es apropiado cuando el usuario puede seleccionar cero, una o varias opciones de un conjunto. A diferencia de JRadioButton, las casillas de verificación no son mutuamente exclusivas.
