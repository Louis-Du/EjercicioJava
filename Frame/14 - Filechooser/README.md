# Selector de Archivos (File Chooser)

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que demuestra el uso del selector de archivos (JFileChooser) en Java Swing. Este componente proporciona diálogos estándar para abrir y guardar archivos, navegando por el sistema de archivos.

## Conceptos Practicados

- Componente JFileChooser
- Diálogos de apertura de archivos
- Diálogos de guardado de archivos
- Filtros de extensiones de archivo (FileFilter)
- Obtención de archivos seleccionados
- Manejo de rutas de archivos (File)
- Validación de selección de usuario
- Gestión de directorios por defecto

## Funcionamiento

La aplicación presenta botones que al ser presionados abren diálogos de JFileChooser. El usuario puede navegar por el sistema de archivos, seleccionar archivos o carpetas, y confirmar o cancelar la selección. La aplicación procesa el archivo seleccionado según sea necesario.

## Notas

JFileChooser es esencial en aplicaciones que requieren que el usuario seleccione archivos para abrir, guardar o procesar. Proporciona una interfaz consistente con el sistema operativo.
