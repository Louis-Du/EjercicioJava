# ScrollPane (Ejercicio 2)

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio adicional que profundiza en el uso de JScrollPane, explorando configuraciones avanzadas y casos de uso específicos. Complementa el ejercicio anterior con técnicas más complejas de gestión de scroll.

## Conceptos Practicados

- Configuración avanzada de JScrollPane
- Políticas de visualización de barras (ALWAYS, AS_NEEDED, NEVER)
- Personalización de viewports
- Sincronización de múltiples scroll panes
- Encabezados de columna y fila
- Control programático del scroll
- Optimización de rendimiento con contenido extenso

## Funcionamiento

La aplicación demuestra técnicas avanzadas de scroll, incluyendo la configuración precisa de cuándo y cómo se muestran las barras de desplazamiento, y cómo controlar la posición del viewport programáticamente.

## Notas

Este ejercicio complementario explora aspectos más detallados de JScrollPane que son útiles en aplicaciones complejas que requieren control fino sobre el comportamiento del desplazamiento.
