# Cuadros Combinados

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que explora el uso de listas desplegables (JComboBox) en Java Swing. Los combo boxes proporcionan una forma eficiente de presentar múltiples opciones en un espacio reducido.

## Conceptos Practicados

- Componente JComboBox
- Listas desplegables de opciones
- Manejo de eventos de selección (ActionListener, ItemListener)
- Obtención del elemento seleccionado (getSelectedItem)
- Población de combo boxes con datos
- Manejo de índices de selección

## Funcionamiento

La aplicación presenta uno o más combo boxes que muestran una lista de opciones cuando el usuario hace clic. El usuario puede seleccionar un elemento de la lista, que se muestra en el campo del combo box.

## Notas

JComboBox es ideal para presentar una lista de opciones predefinidas de forma compacta. Es especialmente útil cuando hay más de 5-6 opciones que ocuparían demasiado espacio como radio buttons.
