# Cuadro de Diálogo

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que demuestra el uso de ventanas de diálogo (JDialog) y diálogos predefinidos (JOptionPane) en Java Swing. Los diálogos son ventanas secundarias que solicitan información o muestran mensajes al usuario.

## Conceptos Practicados

- Componente JDialog
- Diálogos modales y no modales
- JOptionPane para mensajes simples
- Tipos de diálogos (mensaje, confirmación, entrada, opción)
- Personalización de diálogos
- Manejo de resultados de diálogo
- Posicionamiento relativo al frame principal

## Funcionamiento

La aplicación presenta botones que abren diferentes tipos de diálogos: mensajes informativos, advertencias, errores, confirmaciones (Sí/No), diálogos de entrada de texto, y diálogos personalizados con componentes específicos.

## Notas

Los diálogos son esenciales para la interacción con el usuario en aplicaciones de escritorio. JOptionPane simplifica la creación de diálogos comunes, mientras que JDialog permite personalización completa.
