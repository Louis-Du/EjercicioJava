# Evento del Ratón

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que explora el manejo de eventos del ratón (MouseListener, MouseMotionListener) en Java Swing. Demuestra cómo responder a diferentes acciones del ratón como clics, movimientos y arrastre.

## Conceptos Practicados

- Interfaz MouseListener
- Interfaz MouseMotionListener
- Eventos: mouseClicked, mousePressed, mouseReleased
- Eventos: mouseEntered, mouseExited
- Eventos: mouseMoved, mouseDragged
- Obtención de coordenadas del ratón
- Diferenciación entre botones del ratón
- Detección de clics simples y dobles

## Funcionamiento

La aplicación responde a las acciones del ratón mostrando información sobre el evento o realizando acciones específicas. Puede mostrar coordenadas, detectar áreas de clic, permitir dibujo con el ratón, o implementar funcionalidades de arrastrar y soltar.

## Notas

El manejo de eventos del ratón es fundamental para crear interfaces interactivas y responsivas. Es especialmente importante en aplicaciones de dibujo, juegos y sistemas de manipulación directa.
