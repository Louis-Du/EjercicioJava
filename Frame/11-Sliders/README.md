# Deslizadores (Sliders)

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que explora el uso de deslizadores (JSlider) en Java Swing. Los sliders proporcionan una forma intuitiva de seleccionar valores dentro de un rango mediante un control deslizante.

## Conceptos Practicados

- Componente JSlider
- Orientación horizontal y vertical
- Configuración de rango (mínimo, máximo)
- Marcas principales y secundarias (major/minor ticks)
- Etiquetas de valores
- Manejo de eventos de cambio (ChangeListener)
- Configuración visual (paintTicks, paintLabels)

## Funcionamiento

La aplicación presenta uno o más sliders que el usuario puede manipular arrastrando el deslizador. Los sliders pueden mostrar marcas y etiquetas para facilitar la selección precisa de valores. Los cambios generan eventos que permiten actualizar la interfaz en tiempo real.

## Notas

JSlider es ideal para permitir la selección visual de valores numéricos dentro de un rango. Es más intuitivo que campos de texto para muchas aplicaciones, especialmente en ajustes y configuraciones.
