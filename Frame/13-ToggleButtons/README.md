# Botones de Alternancia (Toggle Buttons)

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que explora el uso de botones de alternancia (JToggleButton) en Java Swing. Los toggle buttons son botones que mantienen su estado (presionado/no presionado) y cambian de apariencia cuando se hace clic en ellos.

## Conceptos Practicados

- Componente JToggleButton
- Manejo de estados de botón (presionado/no presionado)
- Verificación de estado (isSelected)
- Manejo de eventos (ActionListener, ItemListener)
- Agrupación con ButtonGroup
- Cambio visual de estado
- Uso como control de activación/desactivación

## Funcionamiento

La aplicación presenta botones de alternancia que cambian su estado visual al hacer clic. A diferencia de botones normales, los toggle buttons permanecen "presionados" hasta que se hace clic nuevamente, funcionando como interruptores.

## Notas

JToggleButton es útil para representar opciones que pueden estar activadas o desactivadas. Pueden usarse individualmente o agrupados para crear comportamientos similares a radio buttons.
