# Modelos de Cuadro de Lista

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio avanzado que explora el uso de modelos de datos personalizados para JList. Demuestra cómo implementar ListModel para controlar el contenido, ordenamiento y actualización de listas.

## Conceptos Practicados

- Interfaz ListModel
- Implementación de DefaultListModel
- Patrón Model-View-Controller (MVC)
- Notificación de cambios (ListDataListener)
- Actualización dinámica de contenido
- Manipulación programática de elementos
- Ordenamiento y filtrado de datos

## Funcionamiento

La aplicación utiliza modelos personalizados para gestionar los datos de JList. Los modelos permiten operaciones como agregar, eliminar, modificar y reorganizar elementos, con sincronización automática entre el modelo y la vista.

## Notas

Los modelos de lista son esenciales para aplicaciones que requieren manipulación dinámica de datos en listas. Separan la lógica de datos de la presentación, facilitando el mantenimiento.
