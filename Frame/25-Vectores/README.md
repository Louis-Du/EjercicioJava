# Vectores

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que explora el uso de vectores (arrays dinámicos) en combinación con componentes Swing, típicamente tablas (JTable). Demuestra cómo gestionar colecciones de datos dinámicas en interfaces gráficas.

## Conceptos Practicados

- Manejo de vectores y arrays dinámicos
- Componente JTable
- Modelos de tabla (TableModel)
- Actualización dinámica de datos tabulares
- Inserción y eliminación de filas
- Ordenamiento de datos
- Selección de filas y columnas
- Renderizado personalizado de celdas

## Funcionamiento

La aplicación presenta una tabla o interfaz que gestiona datos en vectores. Los usuarios pueden agregar, modificar y eliminar elementos, con actualización automática de la visualización en la interfaz gráfica.

## Notas

La combinación de vectores con JTable es común en aplicaciones que manejan datos tabulares dinámicos. Vector (o ArrayList) proporciona la estructura de datos flexible necesaria para gestionar colecciones variables.
