# Cuadros de Lista

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que demuestra el uso de listas (JList) en Java Swing. Los componentes JList permiten mostrar múltiples elementos en un área de lista con soporte para selección simple o múltiple.

## Conceptos Practicados

- Componente JList
- Modelos de lista (ListModel)
- Selección simple y múltiple
- Manejo de eventos de selección (ListSelectionListener)
- Obtención de elementos seleccionados
- Scroll automático con JScrollPane
- Población de listas con datos

## Funcionamiento

La aplicación presenta una o más listas que muestran colecciones de elementos. El usuario puede seleccionar uno o varios elementos según la configuración del modo de selección de la lista.

## Notas

JList es apropiado cuando se necesita mostrar y permitir la selección de elementos de una colección. A diferencia de JComboBox, JList muestra múltiples elementos simultáneamente.
