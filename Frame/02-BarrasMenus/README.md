# Barras de Menús

## Tipo
Interfaz Gráfica (Frame/Swing)

## Descripción

Ejercicio que demuestra la implementación de barras de menú (JMenuBar) y menús desplegables (JMenu) en aplicaciones Swing. Explora la creación de sistemas de menú jerárquicos con múltiples opciones y submenús.

## Conceptos Practicados

- Componente JMenuBar
- Componente JMenu
- Creación de elementos de menú (JMenuItem)
- Estructura jerárquica de menús
- Manejo de eventos en elementos de menú
- Atajos de teclado (mnemonics)
- Organización de funcionalidades en menús

## Funcionamiento

La aplicación presenta una barra de menú en la parte superior de la ventana con diferentes menús desplegables. Cada menú contiene opciones que pueden ejecutar acciones específicas cuando son seleccionadas.

## Notas

Las barras de menú son un elemento estándar en aplicaciones de escritorio, proporcionando una forma organizada de acceder a todas las funcionalidades del programa.
