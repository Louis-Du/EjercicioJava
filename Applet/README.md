# Aplicaciones con Interfaz Gráfica (Applet)

## Descripción

Esta carpeta contiene aplicaciones Java con interfaz gráfica desarrolladas inicialmente como Java Applets y posteriormente adaptadas utilizando Swing. Los proyectos implementan funcionalidades interactivas que combinan entrada de datos, validación y cálculos en tiempo real.

## Conceptos Practicados

Los ejercicios en esta sección cubren:

- Diseño de interfaces gráficas con Swing
- Componentes básicos (JFrame, JPanel, JButton, JTextField, JLabel)
- Manejo de eventos (ActionListener)
- Validación de entrada en formularios
- Cálculos matemáticos y conversiones
- Diseño de layouts
- Presentación de resultados en interfaz gráfica

## Proyectos

1. **AnalizadorDeTriangulos**: Aplicación que clasifica triángulos según sus lados y calcula ángulos internos
2. **ConvertidorDias**: Herramienta de conversión de días a horas, minutos y segundos
3. **GeneradorDeFacturas**: Sistema básico de generación de facturas con cálculo automático de totales

## Ejecución

Los proyectos pueden ejecutarse de dos formas:

1. Desde el IDE (NetBeans/Eclipse) ejecutando la clase principal
2. Desde línea de comandos compilando y ejecutando la clase que contiene el método `main`

Cada aplicación presenta una interfaz gráfica intuitiva con campos de entrada y botones para interactuar con la funcionalidad.

## Nota

Aunque algunos proyectos fueron desarrollados originalmente como Applets, han sido adaptados para funcionar como aplicaciones Swing estándar debido a la deprecación de la tecnología Applet en versiones modernas de Java.
